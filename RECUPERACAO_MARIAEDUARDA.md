1. O que é uma issue e para que serve no desenvolvimento de software?
Uma issue é uma forma de registrar tarefas, problemas ou sugestões dentro de um projeto não GitHub. Ela serve para organizar o trabalho da equipe e acompanhar o que ainda precisa ser feito.

2. O que é um Pull Request e por que ele é importante?
Um Pull Resquest é um pedido para juntar as alterações feitas em uma branch principal do projeto. Antes de aceitar essas mudanças, outras pessoas podem revisar o código e sugerir melhorias. Isso ajuda a evitar erros e garante que o projeto continue funcionando corretamente.

3. O que é separação de camadas no código?
A separação de camadas consiste em dividir o programa em partes com responsabilidades diferentes. O arquivo 'app py' normalmente cuida da interface e dá lógica principal da aplicação, enquanto o 'repository.py' é responsável por acessar e manipular os dados. Essa organização deixa o código mais limpo, facilita a manutenção e torna o desenvolvimento em equipe mais simples.
